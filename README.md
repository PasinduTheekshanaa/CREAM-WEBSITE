🍦 CREAM-WEBSITE

A modern and responsive ice cream website developed using HTML, CSS, and JavaScript. This project showcases a simple static website with multiple pages and user-friendly design.

🚀 Features
Responsive and clean user interface
Multiple pages: Home, About, Menu, Contact
Interactive navigation and layout
Attractive design using CSS styling
Fast loading static website
🛠️ Technologies Used
HTML5
CSS3
JavaScript
